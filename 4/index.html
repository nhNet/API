Info: Has to be done online.

1. Copy the following into your <head> tag:

<!-- Importing NH Net Google SignIn -->
<meta name="google-signin-scope" content="profile email">
<meta name="google-signin-client_id" content="398993796104-lq9k21a411mnehe5p94brocp3rs72dr5.apps.googleusercontent.com">
<script src="https://apis.google.com/js/platform.js" async defer></script>
<!-- Finished importing HuamanScripts -->

2. Copy the following where the SignIn button should be (<body> tag):

<!-- This is the SignIn button from the NH Net API -->
<div style="margin-left:100px;" class="g-signin2" data-onsuccess="onSignIn" data-onfailure="onFail">
<!-- This is the end of the SignIn button from the NH Net API -->
