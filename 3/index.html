NH Net API

A free API made for common library implementation into personal websites. Please credit me for my hard work!!!


API Commands:

Copy a command into your JS file or <script> tag:

// Open up Twitch. Requires the user to type in a stremer name. If there are no attributes say "".
OpenTwitch(/*streamer      ,      attributes*/);

// Open up Youtube. Requires the user to type in a video id. If there are no attributes say "".
OpenYouTube(/*video id (e.g. "z6oRgnUMfpg")      ,      attributes*/);

// Add an iframe containing a website in a fullscreen mode with less code.
OpenWebinWeb(/*"https://nhnet.github.io"*/);

// Add NH Net Browser into your webpage.
OpenNHNetBrowser();

// Add NH Net Browser Login into your webpage.
OpenNHNetLogin();

// Setup NH Net WebDev in your website!
setupWebDev();
